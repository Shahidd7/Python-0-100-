# 🚀 Python Developer Portfolio

> **From beginner to advanced — learning Python by building real software, not by memorizing syntax.**

This repository documents my journey of becoming a Python developer through **10 progressively challenging projects**. Every project introduces new concepts naturally while reinforcing previous ones, following the same learning approach used by professional software engineers.

Instead of copying tutorials, every application is designed to be built **feature by feature**, encouraging problem solving, debugging, refactoring, and writing maintainable code.

# 📈 Learning Philosophy

Every project in this repository follows the same engineering workflow:

1. Understand the problem.
2. Break it into small milestones.
3. Build one feature at a time.
4. Test thoroughly.
5. Debug independently.
6. Refactor for readability.
7. Add advanced features.
8. Document the project.

This approach emphasizes **thinking like a software engineer** instead of copying code from tutorials.

# 💻 Getting Started

Clone the repository:

```bash
git clone https://github.com/Shahidd7/python-developer-portfolio.git
cd python-developer-portfolio
```

Create a virtual environment:

```bash
python -m venv venv
```

Activate it:

**Windows**

```bash
venv\Scripts\activate
```

**macOS/Linux**

```bash
source venv/bin/activate
```

Install required packages:

```bash
pip install requests SpeechRecognition pyttsx3 sounddevice numpy
```

Run any project:

```bash
cd 07_weather_dashboard
python weather.py
```
